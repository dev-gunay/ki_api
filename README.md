Kleines Webprojekt, das eine KI über eine API anbindet, um auf Benutzereingaben automatisch Texte zu generieren.

Die Anwendung besteht aus einem einfachen Frontend (HTML & CSS) sowie einem PHP-Backend, das die Anfrage an die KI-API weiterleitet und die Antwort ausgibt.

## Funktionen
- Texteingabe über ein Formular
- Weiterleitung der Anfrage an eine KI-API
- Anzeige der generierten Antwort
- Einfaches, übersichtliches UI

## Technologien
- HTML
- CSS
- PHP
- KI API

## Hinweis
Dieses Projekt dient als Lern- und Demo-Projekt zur Nutzung einer KI über eine API.
